Not sure which version to download?

macOS: Choose Apple Silicon (ARM64) for M1, M2, M3, or M4 Macs (2020 or newer). Choose Intel (x86_64) for older Macs with an Intel Core processor.

To check: Apple menu → About This Mac. “Apple M…” = ARM64. “Intel Core…” = x86_64.

Windows: Download the Windows installer. It supports all modern Intel and AMD processors (x64).
